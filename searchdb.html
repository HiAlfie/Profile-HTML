(?) Our wildcard system uses "%" and "_". You can use % to represent any number of any characters, including none at all. The _ character stands for exactly one of any character. For example, use 'abc%' to find any entry starting with 'abc', and 'a__' for any entry where 'a' is the first of exactly 3 characters.
